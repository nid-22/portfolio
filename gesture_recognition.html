Smart TV Gesture Recognition

Introduction

This project, a collaborative effort between IIITB and Upgrad, aims to revolutionize how users interact with smart TVs. By focusing on the development of a gesture recognition feature, this initiative enables smart TVs to understand and respond to five distinct user gestures. This functionality allows users to control their TVs effortlessly without the need for a remote. Utilizing a 3D Convolutional Network (Conv3D), the system is adept at processing and interpreting spatial-temporal patterns within video sequences of user gestures, making interaction intuitive and futuristic.

Problem Statement

The core challenge addressed by this project is the accurate recognition of user gestures to control TV operations, through the analysis of sequences of RGB images. These images are processed by a sophisticated Conv3D model that interprets the gestures, transforming the way we interact with our televisions.

Dataset

The foundation of this project lies in its dataset, comprising video sequences where each sequence represents a unique gesture performed by the user. Each video consists of a sequence of 30 RGB frames, each frame with a resolution of 100x100 pixels. This forms a 4-dimensional tensor input shape that the Conv3D model processes, emphasizing the model's ability to handle complex spatial-temporal data.

Model Architecture

The project employs a sophisticated 3D Convolutional Network architecture designed specifically for processing and understanding the intricate patterns in video sequences. Below are the key components of our model architecture:

  • Conv3D Layers: The backbone of our model, these layers are crucial for extracting spatial-temporal features from the video sequences. By convolving through both space and time, Conv3D layers capture the dynamics of user gestures.
  • Activation Functions: ReLU (Rectified Linear Unit) is employed following each Conv3D layer to introduce non-linearity into the model. This function is essential for enabling the model to learn complex patterns and relationships in the data.
  • Pooling Layers: Specifically, MaxPooling3D layers are utilized to reduce the dimensionality of the feature maps, thus extracting the most dominant features and reducing computational load.
  • Fully Connected Layers: These layers serve as the decision-making component of the architecture, classifying the extracted features into one of the five gesture categories.
  • Output Layer: The final layer uses a softmax activation function to output a probability distribution over the five classes. This probabilistic approach allows the model to express certainty in its predictions, facilitating more reliable gesture recognition.

This comprehensive model architecture ensures that the Smart TV Gesture Recognition Project not only meets but exceeds the requirements for creating an interactive and user-friendly smart TV experience.
