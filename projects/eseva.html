Eseva - Government Scheme Finder

Check on Github

Project Overview

Eseva is a government scheme finder application designed to connect users with relevant government schemes. The project aimed to simplify the process of discovering and applying for various government initiatives.

Key Features

1. Sign Up for Organisation

2. Sign Up for Individual

3. Landing Page

4. Landing Page After Login

5. Scheme page

The landing page after login features a dynamic sidebar that allows users to filter government schemes based on various criteria. Users can filter schemes by industry, sector, gender, age group, etc., providing a personalized experience.

Card Representation

The landing page displays scheme cards, each representing a specific government initiative. The cards are differentiated based on the types of schemes, making it easy for users to identify and explore relevant opportunities.
