Project Overview

This project implements a full-stack Retrieval-Augmented Generation (RAG) system for fashion product search using the Myntra dataset. It supports semantic query handling, vector-based retrieval, cross-encoder reranking, and final response generation using a language model.

Layers Implemented:
  • Embedding Layer: Vectorized product metadata using OpenAI and SentenceTransformer embeddings.
  • Search Layer: ChromaDB for top-k retrieval and cross-encoder re-ranking.
  • Generation Layer: Used Zephyr-7B for LLM-based answer generation with dynamic prompting.

Technologies Used

  • Sentence Transformers
  • Hugging Face Transformers
  • ChromaDB
  • OpenAI (optional)
  • Pandas, PIL, Matplotlib

Key Features

  • Semantic search over product descriptions
  • Multi-model experimentation (MiniLM, OpenAI, Zephyr)
  • Dynamic prompt construction with reranked context
  • Image and metadata display for top results
